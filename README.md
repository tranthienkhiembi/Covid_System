### Yêu cầu chương trình:
1. Cài đặt nodejs.
2. Cài đặt và tạo 2 cơ sở dữ liệu về Covid, Thanh toán sử dụng Postgres Sql.
### Cách chạy chương trình:
```bash
  Kết nối 2 hệ thống với 2 cơ sở dữ liệu đã được tạo

  npm install

  npm run dev
```
